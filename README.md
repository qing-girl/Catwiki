一、项目概要

此项目是一个vue基础项目，数据纯属模拟测试，图片和数据来源于网络。



Vue版本：2.9.6



开发工具：VScode



主要插件：使用了饿了么公司的移动端插件[Mint-Ui](https://mint-ui.github.io/docs/#/zh-cn2 "Mint-Ui")
